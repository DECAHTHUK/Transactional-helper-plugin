Detects methods annotated with @Transactional(propagation = Propagation.NEVER) that are called within an active transaction, which can lead to unexpected behavior or errors.

This inspection checks whether a method marked with Propagation.NEVER is executed while a transaction is active. If a transaction is already running when the method is called, the inspection highlights the annotation and suggests removing the transaction from the calling context.

Example: 


    @Transactional(propagation = Propagation.NEVER)
    void logOperation() {
        // Logging operation that must not be in a transaction
    }

    @Transactional
    void process() {
        logOperation(); // This will cause an error because a transaction is active
    }
